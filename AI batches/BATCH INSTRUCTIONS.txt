THIS SCRIPT USES THE FFMPEG AI FILTER OF A 3RD PARTY TOOL FOR IMG ENHANCEMENT.

ALL THE EXTRACTED FRAMES ARE ENHANCED ONE BY ONE AND THEN ARE COMBINED AGAIN, 
AT THE END, THE SOUNDTRACK IS ADDED BACK.

YOU NEED TO HAVE YOUR FFMPEG BUILT WITH SOME AI FILTER (BC THAT IS PRETTY
CHALLENGING TO DO, I AM USING ONE OFF-THE-SHELF, WHICH COMES FROM A 3RD PARTY)

YOU EXTRACT THE FRAMES TO AN INPUT FOLDER ON YOUR COMPUTER.

YOU THEN CALL THE BATCH SCRIPT AFTER SETTING THE REQUIRED PARAMETERS IN THE BATCH.

THE BATCH WILL ITERATE OVER EACH FRAME, IN THE SEQUENCE THEY WERE EXTRACTED. BC THIS
PROCESS IS PRONE TO FAILURE, IT CALLS ITSELF IF ANY OUT-OF-MEMORY OR OTHER BUG HAPPENS.
IT RUNS UNTIL ALL FRAMES IN THE INPUT FOLDER HAVE BEEN ENHANCED.

BC SOME OF THESE PARAMETERS ARE HARDCODED IN THE BATCH FILE, YOU NEED TO CHANGE THEM
TO REFLECT YOUR ENVIRONMENT.

THE LOCATION OF THE FFMPEG BUILT WITH THE TVAI FILTER IS ALSO HARD-CODED IN THE BATCH.

GOOD LUCK!
